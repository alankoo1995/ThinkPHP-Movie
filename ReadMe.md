# Admin-System-based-on-ThinkPHP
a basic admin system

使用前请确保已安装好环境：
- Apache 2.2以上
- MySQL 5.0以上

基础设置：
- 网站首页路径设置到/public/
- 后台登录界面：/admin/login

系统的API：
- 数据库表：
  - 文章管理 bk_pages
  - 管理员 bk_admin
- 前端接口：
  - 文章列表： {volist name="pagesRes" id="pagesList"}
  - 缩略图存储路径： /public/uploads
